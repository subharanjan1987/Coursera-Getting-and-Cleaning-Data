    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-getting-and-cleaning-data-course-project" class="anchor" href="#getting-and-cleaning-data-course-project" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting and Cleaning Data Course Project</h1>

<p>This file describes how run_analysis.R script works.</p>

<ul>
<li>First, unzip the data from <a href="https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip">https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip</a> and rename the folder with "data".</li>
<li>Make sure the folder "data" and the run_analysis.R script are both in the current working directory.</li>
<li>Second, use source("run_analysis.R") command in RStudio. </li>
<li>Third, you will find two output files are generated in the current working directory:

<ul>
<li>merged_data.txt (7.9 Mb): it contains a data frame called cleanedData with 10299*68 dimension.</li>
<li>data_with_means.txt (220 Kb): it contains a data frame called result with 180*68 dimension.</li>
</ul></li>
<li>Finally, use data &lt;- read.table("data_with_means.txt") command in RStudio to read the file. Since we are required to get the average of each variable for each activity and each subject, and there are 6 activities in total and 30 subjects in total, we have 180 rows with all combinations for each of the 66 features. </li>
</ul>
